# YouTube Trend Analysis Project
**Created by Sathwik Rajaboina**

## 📊 Overview
Power BI dashboard analyzing YouTube video performance to identify trends in engagement, virality, and optimal upload timing.

## ⚙️ Tools & Data
- Tool: Power BI
- Dataset: YouTube video metadata (views, likes, comments, categories, publish time)
- KPIs:
  - Total Views: 22M
  - Average Views: 768.48K
  - Average Engagement Rate: 5.62%
  - Average Viral Score: 3.55%

## 🎯 Insights
- Entertainment & Music dominate average views.
- Gaming and Autos & Vehicles show moderate engagement.
- Late evening uploads (18:00–23:00) on Wednesdays & Sundays yield highest engagement.
- Viral score averages 3.55%, showing moderate spread potential.

## 📈 Impact
Helps creators optimize upload timing and content strategy for maximum reach.
